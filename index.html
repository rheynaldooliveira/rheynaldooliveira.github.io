O que é um Grid System?

Um grid system é uma estrutura baseada em colunas e linhas que ajuda a organizar o conteúdo de forma consistente, alinhada e responsiva. Este guia explora um sistema de 12 colunas construído com display: flex, seguindo os princípios modernos de desenvolvimento web.

Abordagem Mobile-First

Todos os estilos começam com o layout para dispositivos móveis (menor breakpoint). À medida que a tela aumenta, usamos @media (min-width: ...) com unidades em em para adicionar melhorias — nunca para "corrigir" o mobile.

Isso garante desempenho, acessibilidade e uma experiência progressiva.

Conceitos-Chave

Breakpoints em em

Usamos em (ex: 48em = 768px) porque respeitam a preferência do usuário por tamanho de fonte, tornando o layout mais acessível.

Gutter com Padding + Margem Negativa

O gutter (espaço entre colunas) é criado com:

  • padding-left/right nas colunas
  • margin-left/right: -half-gutter na linha (.row)

Isso mantém o alinhamento perfeito com o container.

Box-sizing: border-box

Essencial para que padding e border façam parte da largura do elemento — evitando quebras no layout.
